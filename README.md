# 📖 1. Giới thiệu đề tài

**Game Bắn Gà Vũ Trụ 2D** là trò chơi được xây dựng bằng ngôn ngữ **Python** kết hợp thư viện **Pygame**.
Game lấy cảm hứng từ tựa game nổi tiếng **Chicken Invaders**, nơi người chơi điều khiển phi thuyền chiến đấu chống lại đội quân gà ngoài hành tinh đang xâm chiếm thiên hà.

### 🎯 Mục tiêu của game

* Rèn luyện kỹ năng lập trình game bằng Python
* Xử lý đồ họa 2D và va chạm vật thể
* Xây dựng gameplay hấp dẫn
* Tạo sản phẩm giải trí mang tính sáng tạo

---

# 🎮 2. Gameplay hấp dẫn

* ✅ Điều khiển tàu vũ trụ bằng **cử chỉ tay**
* ✅ Bắn đạn liên tục tiêu diệt gà địch
* ✅ Gà di chuyển theo đội hình
* ✅ Boss xuất hiện ở màn chơi cao

# ▶️ 6. Cách chạy game

## 1️⃣ Cài đặt thư viện cần thiết

Dự án sử dụng công nghệ **nhận diện cử chỉ bằng camera** kết hợp đồ họa 2D.
Các thư viện cần thiết gồm:

* `pygame`
* `opencv-python`
* `mediapipe`
* `numpy`

### Cài tự động bằng requirements.txt

```bash id="jj45fy"
cd d:\gamebanga2d\chicken_shooting_game
pip install -r requirements.txt
```

### Hoặc cài thủ công

```bash id="0pn1zv"
pip install pygame opencv-python mediapipe numpy
```

---

## 2️⃣ Chạy game

```bash id="9y98nf"
python main.py
```

---

## 3️⃣ Sau khi khởi động

* Game sẽ mở cửa sổ mới
* Camera tự động bật để nhận diện cử chỉ tay
* Dùng tay để điều khiển tàu vũ trụ
* Nhấn **ESC** để thoát game

---

## 📌 Lưu ý

* Máy tính cần có **webcam** hoạt động tốt
* Khuyến nghị dùng **Python 3.10 hoặc 3.11**
* Nếu camera không mở, hãy kiểm tra quyền truy cập camera trong Windows
